# Learning PC

 Aplicación de escritorio para aprender Windows y programación básica de forma interactiva.

---

## 👤 Autor

**Desarrollado por I3lue** - Aprendizaje interactivo para todos.

---

## 📱 Descripción

Learning PC es una aplicación educativa de escritorio que teach computing fundamentos through interactive simulations. Combines un entorno de aprendizaje gamificado with práctica real en un entorno Windows emulado.

**Target users**: Personas que wants aprender Windows, Atajos de teclado, y básica programación desde cero.

---

## 🎯 Características Principales

### Aprendizaje Interactivo
- Simulaciones practicas en un Windows emulado (VSCode, Explorador, etc.)
- Evaluaciones automatic con feedback inmediato
- Tutor step-by-step con validacion

### Sistema de Progreso
- Tracking automatico de completado por nivel
- Progreso global y por categoría
- Badges y logros

### Interfaz
- Diseño Fluent Windows 11
- Modo claro/oscuro
- Color secundario personalizable
- Tema adaptativo para usuarios
