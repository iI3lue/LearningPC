¿Cómo funciona un Intérprete?

El Intérprete es el componente del IDE que traduce y ejecuta tu código Python en tiempo real.

  • 1. Lectura: Lee tu código línea por línea
  • 2. Análisis: Verifica la sintaxis (estructura)
  • 3. Ejecución: Convierte a instrucciones de máquina
  • 4. Output: Muestra el resultado inmediato

Diferencia con Compilador:

El Intérprete ejecuta línea a línea inmediatamente (Python). Un compilador traduce todo el código antes de ejecutar (C++, Java).
