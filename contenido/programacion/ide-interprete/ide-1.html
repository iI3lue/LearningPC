Que es un IDE?

Un IDE (Entorno de Desarrollo Integrado) es un programa que te ayuda a escribir codigo.

Un IDE tiene:

  • Editor de texto - donde escribes tu codigo
  • Intérprete/Compilador - que ejecuta tu codigo
  • Depurador - para encontrar errores

En este IDE simulado, puedes escribir codigo Python y verlo ejecutarse.
