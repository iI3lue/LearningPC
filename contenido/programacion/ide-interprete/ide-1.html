🤖 ¿Qué es un IDE?

Visual Studio Code - El entorno de desarrollo más popular

📝 IDE = Entorno de Desarrollo Integrado

Un IDE es un programa que agrupa varias herramientas:

  • Editor de código - Escribes tu código con resaltado de sintaxis
  • Intérprete/Compilador - Traduce y ejecuta tu código
  • Depurador - Encuentra y corrige errores
  • Autocompletado - Sugiere código mientras escribes

⚙️ IDE vs Intérprete

Intérprete solo ejecuta código (como Python en consola).

IDE = Editor + Intérprete + Depurador + Más herramientas.
