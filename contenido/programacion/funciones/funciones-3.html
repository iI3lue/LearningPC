📤 Funciones - Return

Python - Valores de retorno

¿Qué es return?

El return devuelve un valor:

  • Finaliza la función
  • Retourna un valor al que llamó
  • Se puede guardar en una variable

✏️ Sintaxis

def suma(a, b):
    return a + b

Usar: resultado = suma(3, 5)

Vamos a usar return.

📤 ¿Qué es return? 1/4

El return devuelve un valor:

  • Finaliza la ejecución de la función
  • El valor vuelve al lugar donde se llamó
  • Se puede guardar en una variable

Sin return, la función retorna None.

✏️ Sintaxis con return 2/4

Definir:

def nombre(params):
    return valor

Usar:

resultado = nombre(args)
  • return devuelve el valor
  • Se guarda en una variable
👆 Crear con return 3/4

Escribe (cada línea en una nueva línea):

def suma(a, b):
    return a + b
resultado = suma(3, 5)
print(resultado)

El return debe estar indentado dentro de la función
