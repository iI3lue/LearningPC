Python - Código reutilizable

¿Qué es una función?

Una función es un bloque de código reutilizable:

  • Se define una vez
  • Se puede llamar muchas veces
  • Ayuda a organizar y reutilizar código

✏️ Sintaxis

def nombre():
    print("Hola")

Usa def + nombre + (): + indentación

Vamos a crear tu primera función.

📤 ¿Qué es una función? 1/4

Una función es código reutilizable:

  • def = definir (crear) la función
  • Nombre: cómo la llamas (ej: saludar)
  • Código: lo que hace cuando la llamas

Se define una vez, se usa muchas veces.

✏️ Sintaxis de función 2/4

Estructura:

def nombre():
    # código indentado
  • def palabra clave
  • nombre() nombre + paréntesis
  • : dos puntos
  • Código con indentación (4 espacios)
👆 Crear la función 3/4

Escribe:

def saludar():
    print("Hola") saludar()

Primero define, luego llama la función
