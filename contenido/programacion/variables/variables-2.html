🔢 Tipos de Datos

Python - str, int, float, bool

Python tiene tipos de datos

Cada valor en Python tiene un tipo:

  • str - Texto (siempre con comillas)
  • int - Entero (sin decimal)
  • float - Decimal (con punto)
  • bool - Bool (True o False)

✏️ Ejemplos

nombre = "Ana" → str
edad = 25 → int
estatura = 1.65 → float
activo = True → bool

Vamos a practicar identificando tipos con type().

🔢 Tipos de Datos en Python 1/4

Python detecta automáticamente el tipo del dato:

  • str (string): texto con comillas
    "Ana", "Hola"
  • int (integer): número entero
    25, 100
  • float: número decimal
    1.65, 3.14
  • bool: True o False
    True, False
📋 Ejemplos de Tipos 2/4

Crea variables de diferentes tipos:

nombre = "Ana" (str: texto)
edad = 25 (int: número)
estatura = 1.65 (float: decimal)
activo = True (bool: verdadero)

Usa type() para ver el tipo:

print(type(nombre))
