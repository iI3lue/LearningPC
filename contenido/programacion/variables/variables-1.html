📦 ¿Qué es una variable?

Python - Conceptos Fundamentals

📦 Una variable es un contenedor

Una variable en Python es como una caja donde guardas un valor.

  • Tiene un nombre (para identificarla)
  • Guarda un valor (dato)
  • El valor puede cambiar (por eso se llama "variable")

✏️ Sintaxis en Python

nombre = valor

Ejemplos:
x = 5 (número)
nombre = "Ana" (texto)
activo = True (verdadero/falso)

📦 ¿Qué es una variable? 1/4

Una variable es un contenedor que guarda un valor en memoria.

  • Nombre: cómo la identificas (ej: x, nombre)
  • Valor: el dato que guardar (ej: 5, "Ana")
  • Tipo: Python lo detecta automáticamente

En Python: No necesitas declarar el tipo.

✏️ Sintaxis en Python 2/4

Para crear una variable usa:

nombre = valor

  • El signo = asigna el valor
  • Texto siempre con comillas
  • Los números van sin comillas

Ejemplos:

edad = 25
nombre = "Maria"
precio = 19.99
